## 🇧🇷 Português

A **Ensigna** é uma landing page corporativa e responsiva desenvolvida sob medida para a especialista Emily Albuquerque. O objetivo principal do projeto é centralizar e apresentar serviços de gestão operacional, financeira e administrativa focados no mercado digital, permitindo que empreendedores deleguem tarefas burocráticas e otimizem o seu tempo estratégico.

### 🚀 Funcionalidades Principais
* **Design Premium & Fluido:** Construído com uma paleta de cores terrosas/sofisticadas e tipografia elegante (`Plus Jakarta Sans` e `Playfair Display`).
* **Calculadora de Planos Interativa:** Uma ferramenta em JavaScript nativo que calcula o orçamento mensal estimado com base nos serviços selecionados pelo utilizador.
* **Integração Direta com o WhatsApp:** Envio automatizado do escopo detalhado de serviços e dados de contacto diretamente para o chat da especialista via API do WhatsApp.
* **Componentes Responsivos:** Navegação otimizada para dispositivos móveis através de um menu de gaveta (drawer menu).
* **Independência de Scripts de Ícones:** Renderização direta de logótipos via vetores SVG estáticos (Bootstrap Icons) para máxima performance de carregamento.

### 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **Tailwind CSS:** Framework utilitário para estilização e design responsivo.
* **JavaScript (ES6+):** Lógica da calculadora, manipulação do DOM do menu móvel e integração de APIs de mensagens.
* **Bootstrap Icons:** Vetores de marca para redes sociais.

## 🇺🇸 English

**Ensigna** is a premium, fully responsive corporate landing page tailored for digital market specialist Emily Albuquerque. The primary goal of this project is to centralize and showcase operational, financial, and administrative management services, enabling entrepreneurs to outsource routine tasks and focus on growth.

### 🚀 Key Features
* **Premium & Fluid Design:** Built with a sophisticated earthy color palette and elegant typography (`Plus Jakarta Sans` & `Playfair Display`).
* **Interactive Plan Calculator:** A native JavaScript tool that instantly estimates monthly investment based on the user's selected services.
* **Direct WhatsApp Integration:** Automated delivery of the selected service scope and contact details directly to the specialist's chat via WhatsApp API.
* **Mobile-First Responsiveness:** Optimized navigation across all devices featuring a dynamic mobile drawer menu.
* **Zero-Script Icon Loading:** Direct SVG vector rendering (Bootstrap Icons) for lightning-fast page loading performance.

### 🛠️ Tech Stack
* **HTML5:** Semantic web structuring.
* **Tailwind CSS:** Utility-first framework for modern styling and responsiveness.
* **JavaScript (ES6+):** Business logic for the calculator, mobile menu DOM manipulation, and messaging integration.
* **Bootstrap Icons:** Asset brand iconography.
